Valorant

Valorant é um jogo de tiro tático 5v5 em primeira pessoa, gratuito, desenvolvido pela Riot Games. Nele, os jogadores e scolhem agentes com habilidades únicas e disputam partidas onde estratégia, mira e trabalho em equipe são essenciais. O objetivo principal é plantar ou desarmar a spike, e cada rodada exige precisão e tática. Com visual estilizado, mecânicas competitivas e forte presença nos eSports, Valorant é um dos FPS mais populares da atualidade.

Tecnologias trabalhadas

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para estruturar o conteúdo de páginas web. Em outras palavras, é a base que define como os elementos de uma página, como texto, imagens, links, e outros, são organizados e exibidos.

CSS

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para estilizar documentos HTML. Ela controla a aparência visual de páginas web, como cores, fontes, layout e outros elementos de design.

JavaScript

JavaScript é uma linguagem de programação de alto nível, frequentemente usada para criar páginas web interativas, mas também utilizada em outros ambientes. É uma linguagem interpretada e baseada em objetos, com funções de primeira classe.
